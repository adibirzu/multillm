# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, in full and without modification.

The canonical text is published at:

- <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
- Plain text: <https://www.contributor-covenant.org/version/2/1/code_of_conduct.txt>

Participation in the MultiLLM Gateway community — issues, pull requests, discussions, chat — implies acceptance of those terms.

## Reporting

Please use the private security disclosure process in [SECURITY.md](./SECURITY.md) for any code-of-conduct violation involving sensitive information.

For non-sensitive reports, open an issue using the appropriate template, or contact the maintainers via the channels listed on the project's repository page.

## Scope

The Code of Conduct applies to all community spaces operated by this project and to public spaces when an individual is representing the project or its community.
